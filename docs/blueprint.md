# **App Name**: CORE Biz Manager

## Core Features:

- User Authentication: Secure user accounts with Supabase Auth (email + password).
- Prompt Console: Central text box for users to input commands in natural language.
- Intelligent Parsing: Rule-based parsing to interpret user prompts for inventory, sales, and metrics, later upgradeable to an AI/NLP tool like the OpenAI API.
- Dashboard KPIs: Display key performance indicators like total sales, profit, and top-selling products.
- Inventory Management: Add, edit, and delete products; track stock levels.
- Sales Recording: Log transactions with details like item, price, quantity, and payment type.
- Insights Engine: Rule-based suggestions based on business data (e.g., low stock alerts).

## Style Guidelines:

- Primary color: Deep Indigo (#3F51B5) for a sense of trust and authority.
- Background color: Light Gray (#F0F2F5), a desaturated version of the primary, provides a neutral backdrop.
- Accent color: Violet (#9C27B0) is used to provide contrast with the primary color, adding a modern twist.
- Body font: 'PT Sans', a humanist sans-serif for both headers and body, modern yet readable.
- Code font: 'Source Code Pro' for displaying code snippets clearly.
- Modern, minimalist icons that clearly represent each feature (e.g., graph for sales, box for inventory).
- Clean and mobile-responsive design using TailwindCSS; card-based layout for dashboard and insights.
- Subtle animations for transitions and feedback to user prompts.
